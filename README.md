# Практика ПМ02: Разработка и изучение API

## Обзор
Данный репозиторий содержит материалы и проекты, выполненные в рамках профессионального модуля **ПМ02**. Основная цель — разработка и исследование бэкенд-компонентов с использованием `AgroChem.API`.

## Структура проекта
Репозиторий организован по принципу разделения на отдельные сессии (итерации) и варианты сборки:

| Папка | Описание |
| :--- | :--- |
| `Сиссия1` ... `Сиссия7` | Папки, последовательно отражающие этапы (сессии) выполнения практических работ. |
| `Сиссия2/AgroChem.API` ... `Сиссия5/AgroChem.API` | Варианты реализации API для разных этапов или разных исполнителей. |
| `AgroChem.API` (в корне) | Финальная или основная версия программного интерфейса. |

## Содержание
- **AgroChem.API**: Исходный код веб-API (предположительно на .NET Core/ASP.NET Core).
- **README.md**: Текущий файл с описанием.
- **Папки СиссияX**: Архивные или промежуточные версии для контроля истории обучения.
